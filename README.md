Wooden_Engine
=============
C++, SDL2
<br>2D game engine

Building steps:
=============

###Linux

Install dependencies:
```
sudo apt-get install g++ make cmake libsdl2-dev libsdl2-image-dev libsdl2-ttf-dev 
```
Build source:
```
$ cd /path/to/build/
$ cmake /path/to/folder/with/cmakelists
```
